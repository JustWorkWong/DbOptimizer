# Task Cards Index

## Purpose

这份索引把 `TASK-*` 拆成可直接派发的独立任务卡。每张任务卡都固定包含目标、依赖、类与方法、文件、接口、数据库、步骤、验证和完成标准。

## Usage

派发单个任务时，最少附带：

1. 当前任务卡
2. [AI_TASK_PACKET_TEMPLATE.md](./AI_TASK_PACKET_TEMPLATE.md)
3. 对应 API 契约文档
4. 对应 workflow / frontend / data 设计文档
